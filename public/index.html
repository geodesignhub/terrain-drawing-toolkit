Geodesign Hub Terrain Viewer

View designs synthesized on Geodesign Hub on a terrain.

Usage Instructions:

  1. Use the Geodeisgn Hub API GET Synthesis JSON call to get a design JSON. Sample GeoJSON is here.
  2. Paste the GeoJSON below
  3. Click "Generate Scene" button

This project uses Cesium library for visualization and design data from Geodesign Hub. Please see the GitHub repository for more information.
